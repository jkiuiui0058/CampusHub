# 前端服务

##  技术栈

- **框架**: Vue 3.5.31
- **构建工具**: Vite 6.2.5
- **Node 版本**: 22 LTS

## 前端结构

frontend/
├── public/           # 静态资源
├── src/
│   ├── assets/       # 样式、图片、字体
│   ├── components/   # 公共组件
│   └── main.js       # 入口文件
│   └── App.vue       # 根组件
├─  package.json
├─  eslint.config.js  # 代码规范
├─  index.html        # 页面入口
└─  vite.config.js    # Vite配置



## 快速开始

安装依赖
```
npm install
```
启动开发环境
```
cd frontend
npm run dev
```
