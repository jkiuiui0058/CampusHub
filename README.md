# CampusHub 校园互助服务平台

## 一、项目简介

CampusHub 是一个面向高校场景的校园互助服务平台，旨在解决学生日常互助需求信息分散、匹配效率低的问题。

用户可以在平台上发布互助需求（如快递代取、学习辅导、二手交易、活动组队等），其他用户可以浏览并接单完成服务。系统提供需求发布、订单流程管理、评价与信用体系等功能，帮助用户高效完成协作并建立信任关系。

---

## 二、项目目标

本项目基于《软件工程与计算 II》课程要求，完成一个具有基本功能的 Web 应用系统，并在开发过程中引入 AI 工具辅助，提升开发效率与工程实践能力。
